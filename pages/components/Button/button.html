Button

            
                @import url("https://byoticss.netlify.app/pages/components/button/button.css");
                <link rel="stylesheet"
                 href="https://byoticss.netlify.app/pages/components/button/button.css">

Solid Buttons

The solid button comes in three variants btn-primary, btn-secondary and btn-danger. 

            
                <button class="btn btn-primary">Primary</button>
                <button class="btn btn-secondary">Secondary</button>
                <button class="btn btn-danger">Danger</button>

Outline Buttons

The outline button comes in three variants btn-outline-primary, btn-outline-secondary and btn-outline-danger. 

            
                <button class="btn btn-outline-primary">Primary</button>
                <button class="btn btn-outline-secondary">Secondary</button>
                <button class="btn btn-outline-danger">Danger</button>

Icon Buttons

Add btn-icon class to the button and an img tag inside it with the link to the icon. 

            
                <button class="btn btn-icon btn-primary">
                    <img src="..." alt=""/>
                    <span>Add to cart</span>
                </button>

                <button class="btn btn-icon btn-danger">
                    <img src="..." alt=""/>
                    <span>Remove</span>
                </button>
