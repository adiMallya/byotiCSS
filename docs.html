Buttons

Add custom button styles for actions in your page with support for multiple sizes and states. You can use these btn classes on <a> or <button> elements.

Primary Button Styles

Want to make your users click on THAT BUTTON? use primary style buttons.

Primary-text

Secondary Button Styles

If you want user not to pay attention to buttons then use Secondary buttons.

Text Link

Disabled state

Make buttons look inactive by adding the btn-disabled class to any <button> element.

Sizes

Fancy larger or smaller buttons? Add btn-lg or btn-sm class for additional sizes.

Toogle and Floating action Buttons

Use toggle button to switch between two states. Toggle button can be used to switch the theme of your page.
Floating action button triggers the primary action in you app's UI. Customize them to your need by adding the font-awesome icon. This button will be fied to right-bottom corner.

Images

Images can be made responsive to fit the parent's width, and also be customised to be round.

Responsive Image

Add img-res class to make your image fit the width of container. The image height gets adjusted by keeping the aspect ratio same.

Round Image

You can add img-round to make your image shape round.
