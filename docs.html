Disabled state

Toogle and Floating action Buttons

Use toggle button to switch between two states. Toggle button can be used to switch the theme of your page.
Floating action button triggers the primary action in you app's UI. Customize them to your need by adding the font-awesome icon. This button will be fied to right-bottom corner.
